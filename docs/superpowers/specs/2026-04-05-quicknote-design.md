# QuickNote 设计文档

> 轻巧快捷的 Windows 桌面备忘录，全局快捷键呼起，数据完全本地存储。

## 1. 概述

QuickNote 是一个 Windows 桌面端的轻量备忘录应用。核心场景：用户突然有灵感，按下快捷键即可呼出窗口，快速记录后窗口消失，回到之前的工作。数据以 Markdown 文件形式完全保存在本地。

## 2. 技术栈

| 层 | 技术 | 说明 |
|---|---|---|
| 应用框架 | Tauri v2 | Rust 后端 + WebView 前端，打包体积小（~5-10MB） |
| 前端 | Solid.js + Vite | 响应式 UI，极小 bundle |
| 后端 | Rust | 文件 I/O、全局快捷键、系统托盘、窗口管理 |
| 插件 | tauri-plugin-global-shortcut | 全局快捷键注册 |
| 插件 | tauri-plugin-autostart | 开机自启 |
| 序列化 | serde + serde_yaml | 解析 Markdown frontmatter |

不使用额外 UI 组件库，手写 CSS 保持极简。

## 3. 架构

### 3.1 整体架构

单窗口应用，根据用户设置切换"居中输入框"和"侧边栏"两种形态。启动后常驻系统托盘，窗口默认隐藏。

```
┌─────────────────────────────────────┐
│           Tauri Shell               │
│  ┌──────────┐  ┌─────────────────┐  │
│  │ Rust 后端 │  │  Solid.js 前端   │  │
│  │          │  │                 │  │
│  │ - 全局快捷键│  │ - 居中输入框视图  │  │
│  │ - 文件读写  │  │ - 侧边栏视图     │  │
│  │ - 系统托盘  │  │ - 设置页面       │  │
│  │ - 窗口管理  │  │ - 搜索/筛选      │  │
│  │ - 开机自启  │  │                 │  │
│  └──────┬───┘  └────────┬────────┘  │
│         │    Tauri IPC   │          │
│         └───────┬────────┘          │
│                 │                   │
│     ┌───────────▼──────────┐        │
│     │  本地文件系统          │        │
│     │  ~/QuickNote/notes/  │        │
│     │  ~/QuickNote/config/ │        │
│     └──────────────────────┘        │
└─────────────────────────────────────┘
```

### 3.2 Rust 后端职责

- **全局快捷键**：通过 tauri-plugin-global-shortcut 注册/注销快捷键，按下时通过 Tauri event 通知前端
- **窗口管理**：控制窗口的显示/隐藏、位置（居中 or 右侧贴靠）、大小
- **文件 I/O**：笔记的 CRUD 操作——创建、读取、更新、删除 Markdown 文件
- **系统托盘**：托盘图标 + 右键菜单
- **开机自启**：通过 tauri-plugin-autostart 管理
- **配置管理**：读写 config.json

### 3.3 前端职责

- **UI 渲染**：居中输入框视图、侧边栏视图、设置页面
- **状态管理**：Solid.js 响应式 signal 管理笔记列表、搜索状态、筛选条件
- **IPC 调用**：通过 Tauri invoke 调用 Rust 端命令

## 4. 数据模型与存储

### 4.1 目录结构

```
~/QuickNote/
├── notes/
│   ├── 2026-04-05_143052_a1b2c3.md
│   ├── 2026-04-05_150030_d4e5f6.md
│   └── ...
└── config.json
```

### 4.2 笔记文件格式

Markdown 文件，YAML frontmatter 存元数据，正文纯文本：

```markdown
---
id: a1b2c3
created: 2026-04-05T14:30:52
tags: [工作, 灵感]
pinned: true
---
用户反馈搜索太慢，考虑给标题和内容加全文索引
```

- 文件名格式：`{日期}_{时间}_{短ID}.md`，方便文件系统排序
- `id`：6 位随机字符串，保证唯一
- `created`：ISO 8601 时间戳
- `tags`：字符串数组，可为空
- `pinned`：布尔值，默认 false

### 4.3 配置文件

`config.json` 存储用户设置：

```json
{
  "hotkey": "Alt+Space",
  "mode": "center",
  "theme": "dark",
  "autostart": true,
  "notesDir": "~/QuickNote/notes"
}
```

| 字段 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| hotkey | string | "Alt+Space" | 全局快捷键 |
| mode | "center" \| "sidebar" | "center" | 呼出模式 |
| theme | "dark" \| "light" | "dark" | 主题 |
| autostart | boolean | true | 开机自启 |
| notesDir | string | "~/QuickNote/notes" | 笔记存储目录 |

## 5. 窗口行为与交互

### 5.1 启动流程

开机自启 → 静默启动 → 常驻系统托盘 → 注册全局快捷键 → 不弹窗，零打扰。

### 5.2 快捷键唤起/隐藏

- 按快捷键 → 窗口显示（根据设置选居中或侧边栏）
- 再按快捷键 / Esc / 点击窗口外部 → 窗口隐藏（hide，非 close）
- 进程常驻，窗口隐藏时不占用可见资源

### 5.3 居中输入框模式

- 无边框透明窗口，约 500×160px，屏幕居中
- 自动聚焦输入框，直接打字
- **Enter** → 保存笔记并隐藏窗口
- **Shift+Enter** → 换行（支持多行输入）
- 输入 `#标签名` 自动识别为标签
- **Ctrl+S** → 切换收藏/置顶

### 5.4 侧边栏模式

- 无边框窗口，约 360px 宽，贴靠屏幕右侧，高度铺满
- 顶部：新建笔记输入框
- 中间：笔记列表（时间倒序，置顶在最前）
- 底部：标签筛选栏
- **Ctrl+F** 或点击搜索图标 → 搜索
- 点击笔记 → 展开编辑/删除

### 5.5 托盘菜单

- 显示/隐藏窗口
- 切换模式（居中 ↔ 侧边栏）
- 设置
- 退出

## 6. 搜索与筛选

### 6.1 索引策略

- 启动时扫描 `notes/` 目录，将所有笔记 frontmatter + 正文加载到内存
- 搜索时对正文做模糊字符串匹配，结果按相关度 + 时间排序
- 文件变更时增量更新索引

### 6.2 筛选方式

- **按标签**：点击标签或输入 `#标签名`
- **按收藏**：只看置顶/收藏的笔记
- 搜索和筛选可叠加（如：搜索"咖啡" + 筛选 #生活）

### 6.3 性能

千条笔记级别，纯内存索引足够。不引入额外搜索库，简单字符串匹配即可。

## 7. 设置页面

**进入方式：** 托盘右键 → 设置 / 侧边栏齿轮图标

**可配置项：**
- 快捷键：按键录入方式（按下组合键直接识别）
- 呼出模式：居中输入框 / 侧边栏
- 主题：暗色 / 亮色
- 开机自启：开/关
- 笔记目录：自定义存储路径

修改即时写入 `config.json`，快捷键变更时实时重新注册。

## 8. 打包与分发

- 单个 `.msi` 安装包，预计 5-10MB
- 安装后约 15-20MB
- 仅支持 Windows

## 9. 不做的事（Scope 边界）

- 不做云同步
- 不做富文本 / Markdown 渲染预览
- 不做多窗口同时打开
- 不做移动端
- 不做多语言（UI 默认中文）

## 10. 代码规范

- 所有代码文件为关键逻辑、函数、模块写清晰注释
- 注释重点解释"为什么"而非"做了什么"，方便后续 AI 理解和修改
